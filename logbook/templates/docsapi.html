Cara Menggunakan API Data Penangkapan PPN Karangantu

API ini disediakan bagi para programmer yang mungkin membutuhkan data secara real time terkait hasil tangkapan di PPN Karangantu, Banten. API ini sangat sederhana dan hanya memberikan request method GET untuk di fetch sebagai informasi saja. Berikut cara lebih lengkap terkait API Data Penangkapan Ikan PPN Karangantu.

base url

/api/

Fetch Hasil Tangkapan

/api/hasiltangkapan/

Fetch Data Kapal

/api/kapal/

Response Api Kapal : /api/kapal/

Response Api Hasil Tangkap : /api/hasiltangkap/
